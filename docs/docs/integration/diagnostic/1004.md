# GirCore1004

- Title: GObject subclass may not be generic
- Message: GObject subclass must be non-generic. GObject does not support generic types. Use a regular subclass (without SubclassAttribute) or extension blocks to allow type safe access to data.

Invalid code:
```csharp
[Subclass<Gtk.Widget>]
public partial class MyWidget<T>
{
}
```

Solution:
```csharp
[Subclass<Gtk.Widget>]
public partial class MyWidget
{
}

public class MyWdiget<T> : MyWidget
{
}
```
