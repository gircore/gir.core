# GirCore1005

- Title: GObject subclass may not be nested inside a generic type
- Message: GObject subclass must not be declared inside a generic type. Use a regular non-generic outer class instead.

Invalid code:
```csharp
public partial class OuterWidget<T>
{
    [Subclass<Gtk.Widget>]
    public partial class MyWidget
    {
    }
}
```

Solution:
```csharp
public partial class OuterWidget
{
    [Subclass<Gtk.Widget>]
    public partial class MyWidget
    {
    }
}
```
