# GirCore2001

- Title: GTK template class must be annotated with `GObject.Subclass` attribute.
- Message: Adding a `GObject.Subclass` attribute at class level is required to create a GTK widget with a composite template support.

Invalid code:
```csharp
[Gtk.Template<Gtk.AssemblyResource>("MyCompositeTemplate.ui")]
public partial class MyCompositeTemplate { }
```

Solution:
```csharp
[GObject.Subclass<Gtk.Box>]
[Gtk.Template<Gtk.AssemblyResource>("MyCompositeTemplate.ui")]
public partial class MyCompositeTemplate { }
```
