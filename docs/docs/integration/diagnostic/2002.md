# GirCore2002

- Title: GObject subclass must inherit from `Gtk.Widget` for composite template support.
- Message: The `GObject.Subclass` attribute must define a subclass which inherits from `Gtk.Widget`.

Invalid code:
```csharp
[GObject.Subclass<GObject.Object>]
[Gtk.Template<Gtk.AssemblyResource>("MyCompositeTemplate.ui")]
public partial class MyCompositeTemplate { }
```

Solution:
```csharp
[GObject.Subclass<Gtk.Widget>]
[Gtk.Template<Gtk.AssemblyResource>("MyCompositeTemplate.ui")]
public partial class MyCompositeTemplate { }
```
