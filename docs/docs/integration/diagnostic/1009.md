# GirCore1009

- Title: Experimental API: `SetCustomDllImportResolver`
- Message: `SetCustomDllImportResolver` is for evaluation purposes only and is subject to change or removal in future updates. Suppress this diagnostic to proceed.

This API allows to load a custom native binary during runtime. This allows for example to call `MSVC` based binaries. As those binaries must not match the gir files which are used to generate GirCore nugets this is discouraged and should only be used for testing purposes.

> [!WARNING]
> Using this API is unsupported as diagnosing bugs is more difficult and the usage with different binaries is completely untested.

The recommended way is to create GirCore binaries that match the custom build native binaries. Please see the [FAQ](https://gircore.github.io/docs/faq.html#how-can-i-use-gircore-with-a-custom-build-native-binary) for more details on how to create custom GirCore binaries.
