# GirCore2003

- Title: Defining a "Gtk.Connect" attribute requires the containing class to be annotated with "Gtk.Template" attribute.
- Message: A "Gtk.Connect" attribute is only evaluated when the containing class is using Gtk composite templates.

Invalid code:
```csharp
public class MyClass
{
    [Gtk.Connect("my_label")]
    private Gtk.Label _myLabel;
}
```

Solution:
```csharp
[GObject.Subclass<Gtk.Widget>]
[Gtk.Template<Gtk.AssemblyResource>("File.ui")]
public partial class MyWidget
{
    [Gtk.Connect("my_label")]
    private Gtk.Label _myLabel;
}
```
